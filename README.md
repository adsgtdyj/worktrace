# WorkTrace 工作轨迹

桌面时间追踪与任务复盘工具，专为产品经理等长期在电脑前办公的用户设计。
自动记录窗口切换、锁屏离开、任务时长，并在偏离当前任务时弹出确认，最终生成每日可视化复盘报告。

- 当前版本：**v1.0.0（首版）**
- 适用平台：Windows 10 / 11
- 技术栈：Python 3.13 · tkinter · SQLite · pywin32

---

## 核心功能

1. **主面板（W / TRACE）**
   - 340×180 无边框置顶小窗，可拖拽 + 屏幕边缘磁吸
   - 当前任务名、30 格刻度进度条（带呼吸光标）、时间数字 1Hz 冒号闪烁
   - 状态徽章：专注中 / 偏离 XX分 / 暂停 / 空闲
   - 右上角菜单：切换任务、今日复盘、统计、任务列表、切换主题、设置、退出

2. **任务切换弹窗**
   - 检测到当前应用与窗口，展示 APP 名与窗口标题
   - 支持键盘 ↑/↓ 选择已有任务、Enter 确认、R 休息、Esc 取消
   - 底部命令行输入框可新建任务
   - 任务过多时自绘暗色滚动条 + 鼠标滚轮滚动

3. **任务列表窗口**
   - 卡片墙布局，当前任务带 ACTIVE 高亮
   - 每张卡片显示任务名、今日时长、专注轮数
   - 支持改 / 删操作，底部命令行输入框新建任务

4. **统计窗口**
   - KPI 卡片：任务数、总时长、专注时长、效率
   - 本周柱状图 + 月目标环
   - 任务排行 + 日均 / 最长连续统计
   - 日期前后切换，关闭按钮右上角

5. **设置窗口**
   - 专注节奏：专注单元 / 短恢复 / 长恢复
   - 追踪阈值：检测频率 / 离开判定 / 偏离容忍 / 锁屏检测 / 无操作判定
   - 启动选项：完成提示音、窗口置顶、关闭到托盘、静默启动、自动追踪、开机自启
   - 各设置项有悬浮说明，支持恢复默认 / 保存

6. **每日复盘**
   - 退出时自动生成 HTML 报告（`worktrace_review_YYYYMMDD.html`）
   - 通过菜单"今日复盘"在浏览器打开

## 安装与运行

### 方式 A：直接运行源码

1. 安装 Python 3.13+
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行：
   ```bash
   python time_tracker_v2.py
   ```

### 方式 B：一键安装脚本（Windows）

双击 `setup.bat` 安装依赖，再双击 `time_tracker_v2.py` 启动。

> 推荐 Python 路径示例：`C:\Users\<用户名>\.workbuddy\binaries\python\versions\3.13.12\python.exe`
> 无控制台启动可使用 `pythonw.exe`，日志会写入 `worktrace_start.log`。
